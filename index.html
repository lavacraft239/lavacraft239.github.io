WRT

Herramienta rápida para pruebas HTTP y RTMP en Termux/Linux

🚀 Características

  • Ligera y rápida
  • Compatible con HTTP, HTTPS y RTMP
  • Perfecta para Termux o Linux
  • Configuración simple de concurrencia y duración
  • Código abierto

📦 Instalación

git clone https://github.com/lavacraft239/Wrt-benchmark
cd Wrt-benchmark
chmod +x wrt
./wrt -h

💡 Ejemplo de uso

./wrt -c 50 -d 30s https://ejemplo.com
